USER:

Action Method URL Body params Pagination, Search Short review
GET ALL GET api/v1/users Nothing api/v1/users?page=[number of page], no search Returns all existing users in database,
to use this you have to be authorized as admin
GET BY ID GET api/v1/users/:id Nothing No pagination, no search Returns user by id in database,
to use this you have to be authorized as admin
CREATE POST api/v1/users/new login, username,
password, confirmPassword,
fullname, bio, image		 No pagination, no search Creates new user and adds it into database,
to use this you have to be authorized as admin
UPDATE PATCH api/v1/users/:id/update login, username,
password, confirmPassword,
fullname, bio, image		 No pagination, no search Updates user by id and saves changes into database,
to use this you have to be authorized as admin
DELETE DELETE api/v1/users/:id Nothing No pagination, no search Deletes user by id from database,
to use this you have to be authorized as admin
